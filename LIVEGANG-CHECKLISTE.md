# Livegang-Checkliste fuer MPU Safe

WICHTIG: Diese Datei ist nur eine Vorlage. Keine echten Kundendaten, Zugangsdaten, Zahlungsdaten oder Steuerdaten im oeffentlichen Repo speichern.

Diese Checkliste sammelt die Daten, die vor dem technischen Livegang final vom Betreiber bestaetigt oder geliefert werden muessen.

## Betreiber- und Kontaktdaten

- [ ] Vollstaendiger Name oder Firma
- [ ] Rechtsform, falls vorhanden
- [ ] Ladungsfaehige Adresse fuer das Impressum
- [ ] E-Mail fuer Kundenanfragen
- [ ] Telefonnummer
- [ ] Verantwortlicher Betreiber der Website
- [ ] Umsatzsteuer-ID, falls vorhanden

## Digistore24 und Zahlungsabwicklung

- [ ] Digistore24-Account erstellt: Ja / Nein
- [ ] Digistore24 Vendor-ID oder relevante Account-Info
- [ ] Bank- und Auszahlungsdaten liegen im Digistore24-Account: Ja / Nein
- [ ] Gewuenschte Zahlungsarten: Klarna, Kreditkarte, SEPA, Vorkasse
- [ ] PayPal aktivieren oder deaktivieren

## Produkte und Angebotslogik

- [ ] MPU Erste-Hilfe-Koffer direkt kaufen oder nur Anfrage
- [ ] MPU Simulator direkt kaufen oder nur Anfrage
- [ ] 1:1 MPU Coaching direkt kaufen oder nur Erstgespraech
- [ ] Finale Preise bestaetigen
- [ ] Finale Liefer- und Zugriffshinweise bestaetigen
- [ ] Finale externe Checkout-Links bereitstellen
